# How-to-get-outlook-contacts-in-apple-mail
Want Outlook contacts in apple mail, then try BitRecover PST to MBOX converter tool, because movement of Outlook contacts in Apple Mail directly is not possible. Apple Mail supports MBOX format and on the other hand Outlook support PST files, that's why the movement of PST file into MBOX format directly is not possible. Use PST to MBOX converter.
